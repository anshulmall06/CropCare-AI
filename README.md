# CropCare AI

## One-Line Description

An AI-powered application that detects crop diseases from leaf images and provides treatment recommendations to farmers.

## Tech Stack

* Frontend: React.js
* Styling: Tailwind CSS
* Backend: FastAPI
* Database: PostgreSQL
* Authentication: JWT Authentication
* AI API: OpenAI API
* Deployment: Vercel (Frontend) + Render (Backend)
